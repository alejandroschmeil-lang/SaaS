Esta aplicación permite crear, gestionar y visualizar tareas por proyecto usando Streamlit.

## Funcionalidades
- Crear y renombrar proyectos
- Agregar y eliminar tareas
- Visualizar diagramas de Gantt por proyecto y global
- Asignar responsables a tareas

## Requisitos
- Python 3.8+
- Streamlit
- pandas
- plotly

## Instalación
```bash
pip install -r requirements.txt
```

## Uso
```bash
streamlit run main.py
```

Los archivos se guardan automáticamente en la carpeta `/data` como CSV.
